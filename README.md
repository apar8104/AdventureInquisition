This is a sinple Choose Your Own Adventure Game project with a few core features.
- User login system:
  This is a system that will ask for a username and save user data
- Core CYOA gameplay:
  Follow story prompts and forge your own path. Each decision will lead to a custom ending.
- Achievements unlock system:
  When logging in as a user, the users achieved ending will be saved. All acheived endings can be viewed
  and will be color coded based on if the user has unlocked them.
